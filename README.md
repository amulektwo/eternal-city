# THE ETERNAL CITY

THE ETERNAL CITY is a browser-based, walkable 3D golden city that houses a sacred scroll archive — you explore it in first person directly in the browser (WASD or arrow keys to walk, mouse to look, click to interact with scrolls and doorways), with the archive's contents served as static data from the `/data/` directory. It is a fully static site with no build step or server, deployed to GitHub Pages from the `main` branch root of `amulektwo/eternal-city`.
